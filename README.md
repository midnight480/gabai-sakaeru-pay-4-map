# （非公式）がばいサカえーるPay 第4弾マップ

佐賀市が発行する「がばいサカえーるPay [第4弾]」の対象店舗を地図上で検索できる非公式Webアプリです。

## 特徴

- 🗺️ **地図表示**: 全892店舗をLeaflet地図上にマーカー表示
- 🔍 **絞り込み検索**: カテゴリ（24種）・エリア（12地域）・店舗名で検索
- 📋 **一覧表示**: カード型のリスト表示に切替可能
- 📱 **レスポンシブ対応**: スマートフォンでも快適に利用可能

## データソース

店舗データは[公式PDF（令和8年2月27日時点）](https://gabai-sakayell.city.saga.lg.jp/Images/gabai20260227.pdf)をもとに作成しています。

## ローカル開発

```bash
npm run serve
```

## デプロイ

Cloudflare Pages を使用：
```bash
npx wrangler pages deploy docs
```
